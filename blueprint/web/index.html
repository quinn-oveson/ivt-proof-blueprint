Theorem 1 Intermediate Value Theorem

If \(f\) is continuous on the interval \([a, b]\) and \(N\) is a number between \(f(a)\) and \(f(b)\), then there exists a number \(c\) in the interval \((a, b)\) such that \(f(c) = N\).
